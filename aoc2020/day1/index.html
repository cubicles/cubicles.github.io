#Reading inputs
with open('inputs/day1.txt') as f:
    input = f.readlines()
numlist=[]
for i in input:
    numlist.append(int(i.split("\n")[0]))

#First part
def day1part1(input):
    for i in range(len(input)):
        for j in range(i+1,len(input)):
            sum = input[i]+input[j]
            if sum == 2020:
                answer = input[i]*input[j]
    return answer

#Second part
def day1part2(input):
    for i in range(len(input)):
        for j in range(i+1,len(input)):
            for k in range(i+2,len(input)):
                sum = input[i]+input[j]+input[k]
                if sum == 2020:
                    answer = input[i]*input[j]*input[k]
    return answer

First we read the input. I have a folder called "inputs" where i throw in all the puzzles input sources. We start using readlines() to get the lines and we proceed to remove the "\n" by splitting it and saving it in the numlist list. For the first part, i loop 2 variables (i,j) through the list and compare the sum to 2020. If it succeeds in finding it, the sum is saved in the var answer. For the second part, i loop between the list using 3 variables (i,j,k) and apply the same logic as in the first part. In both cases, and as the puzzle states, i return the product of the numbers that add up to the requirement.
